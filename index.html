Election Database Management System

HTML, CSS, Bootstrap, ReactJS, NodeJS, ExpressJS, MongoDB

This project was developed for the local government's management of voters for the 2025 election. It is a voter database management software that enables encoders to search, add, and update voter information. The software includes a filter option, allowing encoders to search by Barangay or by category (such as PWD, Youth, or Senior Citizen). Additionally, it features an admin interface that can track encoder activities and maintain an audit trail.
